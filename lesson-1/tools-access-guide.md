# Доступ до AI-агентів — гайд

> Часте питання: "Чи можна зареєструвати безкоштовний акаунт ChatGPT і запустити Pro trial щоб скачати Codex?"
> Коротко: **Codex CLI вимагає платної підписки. Free ChatGPT не підходить. Trial — як пощастить.**
> Нижче — повний розбір.

## TL;DR

| Tool | Безкоштовний акаунт працює? | Trial доступний? | Що робити |

| **Codex CLI** (OpenAI) | ❌ Ні. Потрібен ChatGPT Plus ($20/міс) або Team/Enterprise | 🟡 Іноді — нові акаунти, не гарантовано в UA | Корпоративна підписка агенції або індивідуальний Plus |

## Codex CLI — детально

### Що працює і не працює з різними планами

| План ChatGPT | Доступ до Codex CLI |
|--------------|--------------------|
| Free | ❌ Ні |
| Plus ($20/міс) | ✅ Так |
| Team ($25/користувач/міс при ≥2) | ✅ Так |
| Enterprise | ✅ Так |
| API-only (без підписки) | 🟡 Через API ключ — інша модель оплати pay-per-use |

### Pro trial — що з ним

- OpenAI періодично пропонує trial Plus (7-14 днів) для нових юзерів
- В UA доступність trial **непередбачувана** — то є, то нема
- Trial автоматично переходить в платну підписку якщо не відмінити
- Перевірити можна тільки створивши акаунт і дойшовши до сторінки upgrade → побачиш чи пропонує trial

### Корпоративна підписка агенції

Якщо ваша агенція має корпоративну ChatGPT Team / Enterprise — команда логіниться під корпоративними логінами і Codex доступний без особистих витрат. Уточніть в адміністратора workspace.

## Claude Code — детально

| План Claude.ai | Claude Code CLI |
|----------------|-----------------|
| Free | ❌ Дуже обмежений. Не підходить |
| Pro ($20/міс) | ✅ Так, з лімітами на день |
| Max ($100-200/міс) | ✅ Без лімітів практично |
| API ключ (без підписки) | ✅ Працює, оплата за токени |

**Pay-per-use API:** для активного користувача $5-30/міс. Без щомісячного committment'у.

## Antigravity — найшвидший безкоштовний шлях

- Google AI Studio → Antigravity IDE
- Free tier із Gemini 2.5 Flash і Pro
- IDE-підхід: працюєш в редакторі, не в CLI
- Файл контексту: `GEMINI.md`

**Обмеження free tier:** є rate limits (кількість запитів на день/хвилину), але для 1-2 годин роботи на день вистачає.

## Що рекомендую команді

### Сценарій 1: Є корпоративна ChatGPT-підписка
→ Усі через **Codex CLI** з корпоративним логіном.

### Сценарій 2: Корпоративної нема, але хочемо однакову екосистему
→ Усі купують **Plus особисто** ($20/міс) і ставлять Codex CLI.

### Сценарій 3: Хочемо безкоштовний старт
→ Усі ставлять **Antigravity** (Google free) — щоб подивитись концепт.
→ Хто кайфує — потім переходить на платну Codex/Claude Code.

### Сценарій 4: Mix'n'match (гнучкий)
- 2-3 людини на Plus / Codex
- 1-2 на Claude Pro / Claude Code
- Решта на Antigravity
→ Всі техніки з курсу працюють однаково — `AGENTS.md` / `CLAUDE.md` / `GEMINI.md` мають однаковий зміст.

**Я б рекомендував Сценарій 1 або 4.** Бо однакова тулза = легше підтримувати і вирішувати проблеми разом.
