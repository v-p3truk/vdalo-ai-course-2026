## A. Обрати агента — 3 варіанти

| Tool | Підписка | Сильні сторони | Кому радимо |
|------|----------|----------------|-------------|
| **Codex CLI** (OpenAI) | ChatGPT Plus $20/міс або корп.підписка агенції | Швидкий, простий старт | Default для більшості — особливо якщо вже є Plus |

**Питання про доступ до Codex через Free ChatGPT** — детально в [`tools-access-guide.md`](tools-access-guide.md). Коротко: **Codex CLI вимагає Plus** (Free не підходить), trial Plus в UA не гарантований. Antigravity — безкоштовна альтернатива.

### Рекомендація
- **Якщо в агенції вже є корпоративна підписка ChatGPT** → Codex CLI (під корпоративним акаунтом)

- **Якщо платити не хочеш** → Antigravity

## C. Створити папку проєкту під клієнта

**Структура (мінімум):**

```
[client-name]/
├── AGENTS.md            ← головний файл контексту (заповнюємо в кроці D)
├── 01_strategy/         ← стратегія, дослідження
├── 03_client_materials/ ← фотки, відео, тексти від клієнта
├── 04_creatives/        ← креативи, банери
├── 05_reports/          ← звіти
└── 06_projects/         ← код, боти, лендінги (якщо є)
```

**Команди (приклад для клієнта "MyClient"):**
```bash
cd ~/Documents  # або куди ти кладеш робочі папки
mkdir -p MyClient/{01_strategy,03_client_materials,04_creatives,05_reports,06_projects}
cd MyClient
touch AGENTS.md
```

> ⚠️ Якщо ти на Claude Code — назви файл `CLAUDE.md` замість `AGENTS.md`.
> На Antigravity — `GEMINI.md`. Зміст один і той самий.

## D. Заповнити AGENTS.md

**Template:** [`../templates/01_brand-context-template.md`](../templates/01_brand-context-template.md)

→ Скопіюй у свою папку (`cp` або через UI) → перейменуй на `AGENTS.md` (або `CLAUDE.md` / `GEMINI.md`) → заповни.

**Що принципово важливо заповнити (а не "все підряд"):**

| Секція | Чому критично |
|--------|---------------|
| **01. Persona** | Без неї агент іде в generic AI-slop |
| **02. Бізнес-контекст** | Маржа, цикл, ринок — впливає на все |
| **03. ICP** | Біль клієнта реальними словами |
| **04. Voice DNA** | 3-5 рис + анти-приклади. Без анти — не працює |
| **05. Vocabulary** | "Завжди" / "ніколи" + як називаємо клієнта |
| **06. Few-shot examples** | 1-2 канонічні приклади > 10 правил |
| **08. Активні кампанії** | Якщо порожньо — агент пише "взагалі", не під поточний момент |

**Що можна пропустити на першому кроці:**
- Минулі кейси (10) — додати потім
- Заборонені теми (09) — додати коли побачиш проблему
- Структурні правила (07) — добре мати, але не блокери

**Принцип:** менше шуму = більше сигналу. **Не клади весь PDF brand book на 80 сторінок.** Тільки те що змінює рішення AI.

## FAQ (попередньо)

**В: А якщо я ніколи не працював з терміналом?**
Терміналу боятись не треба — це лише вікно для команд. Перші 30 хв виглядатиме незвично, потім норм. Якщо стопор — пиши в команду курсу, поясню.

**В: А якщо мій клієнт не маркетинговий, а інший?**
Все працює так само. Структура папки і AGENTS.md універсальні — просто заповнюєш під свій випадок.

**В: Чи треба знати програмування?**
**Ні.** Ми використовуємо AI на природній мові. Терминал — це лише запуск агента.

**В: Скільки коштує?**
Codex/Claude Code — $20/міс кожен. Antigravity — безкоштовно. На цьому етапі досить однієї підписки (або безкоштовного варіанту).

**В: А якщо я плутаюсь — куди звертатись?**
В команду курсу. Не соромся "дурних" питань — їх не існує на цьому етапі.
