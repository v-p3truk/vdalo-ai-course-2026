# Воркшоп Урок 1 — Чеклист для учасника

> Роздаємо в кінці теоретичної частини. 30 хв на заповнення + тестування.

## Крок 1: Оберіть клієнта (2 хв)
Один свій активний проєкт, з яким працюєш щодня.

**Мій клієнт:** _______________

## Крок 2: Заповнити template (15 хв)
Відкрий [01_brand-context-template.md](01_brand-context-template.md) і заповни **мінімум** ці секції:
- [ ] 01. Persona
- [ ] 02. Бізнес-контекст (хоча б 5 рядків)
- [ ] 03. ICP — 1 персона
- [ ] 04. Voice DNA — 3 риси + 3 антириси
- [ ] 05. Vocabulary Rules — 5 слів "так" і 5 слів "ні"
- [ ] 06. Один on-brand + один off-brand приклад

## Крок 3: Створити Custom GPT або Claude Project (5 хв)

**Варіант A — Claude Project:**
1. claude.ai → Projects → New project
2. Назва: "AI асистент [бренд]"
3. Project instructions = вставити заповнений template
4. Готово

**Варіант B — Custom GPT:**
1. chat.openai.com → Explore GPTs → Create
2. Configure → Instructions = template
3. Save → Only me

## Крок 4: Тест 3 робочими задачами (8 хв)

Запусти 3 свої **реальні** робочі запити, які ти зазвичай робиш. Наприклад:

| Запит | Без контексту (раніше) | З контекстом (зараз) |
|-------|------------------------|----------------------|
| "Напиши 5 ідей креативів на ЧП" | | |
| "Що написати в email-розсилці новим підписникам" | | |
| "Зроби бриф креативщику на новий продукт" | | |

## Крок 5: Поділись (бонус)
В чаті команди скинь:
- Один найкращий результат
- Одну несподіванку
- Один момент де AI ще не дотягує — обговоримо на Уроці 2

## Frequently asked

**Q: А якщо template дуже довгий — AI не "проковтне"?**
A: Claude Project / Custom GPT — заковтне 200K+ токенів. Template на 3 сторінки = ~3K токенів. Це 1.5% від ліміту.

**Q: Я заповнив тільки половину секцій — чи варто запускати?**
A: Так. 50% контексту дають 80% покращення. Решту дозаповниш по ходу.

**Q: Чи треба окремий проєкт для кожного клієнта?**
A: Так. Контекст одного бренду шумить для іншого.
