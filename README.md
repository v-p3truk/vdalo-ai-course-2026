# Vdalo AI Course 2026

Внутрішній курс з 3 уроків про AI-агентів для команди агенції **Vdalo**.
Матеріали відкриті — щоб інші AI-агенції / маркетологи могли використати template і структуру.

> 📍 **Аудиторія:** маркетологи / PPC / стратеги які хочуть навчитися працювати з агентськими AI-інструментами (Codex, Claude Code, Antigravity) і вписати це в реальні робочі процеси.
>
> 🎯 **Мета:** не "освоїти конкретний інструмент", а вийти на рівень коли AI — не іграшка для одноразових промптів, а інфраструктура яка працює на тебе.

## Швидкий старт (для самостійного проходження)

1. **Подивитись** Урок 1 на YouTube (1:18 год) — https://youtu.be/l4h2Dp7BK8c
2. **Прочитати** [`lesson-1/homework-pack.md`](lesson-1/homework-pack.md) — план ДЗ
3. **Обрати агента** з [`lesson-1/tools-access-guide.md`](lesson-1/tools-access-guide.md) — Codex / Claude Code / Antigravity
4. **Скопіювати** [`templates/01_brand-context-template.md`](templates/01_brand-context-template.md) у свою робочу папку
5. **Заповнити** template під свого клієнта → перейменувати в `AGENTS.md` (або `CLAUDE.md` / `GEMINI.md`)
6. **Виконати** одну реальну задачу з агентом
7. **Записати** 2 wins + 2 проблеми — щоб було що принести на наступний урок

Орієнтовний час: **1.5–2.5 години** на ДЗ.

## Принципи курсу

1. **Практика > теорія.** Кожен урок = реальний кейс, який можна примінити завтра.
2. **Контекст важливіший за промпт.** Це наскрізна ідея всіх 3 уроків.
3. **Agnostic до інструменту.** `AGENTS.md` (Codex), `CLAUDE.md` (Claude Code), `GEMINI.md` (Antigravity) — однаковий зміст. Принципи переносяться.
4. **Кожен урок має outcome-артефакт** — щось матеріальне що учасник створює сам.
