# Урок 3 — Автономні репорти, аналіз даних, AI-візуал

> **Дата:** п'ятниця, 29 травня 2026
> **Передумова:** Урок 2 завершено — у кожного MCP-and-skills setup
> **Тривалість:** 60-80 хв

## Що людина винесе

Працюючий автономний сценарій щотижневого звіту (сам тягне дані → знаходить патерни → формулює гіпотези) + набір інструментів AI-візуалу для швидких мокапів і креативів.

## План (оrieнтовно)

1. Розбір домашок з Уроку 2 (15 хв)
2. Теорія: автономні агенти, sub-agents, безпека, рівні автономії (10 хв)
3. Практика (50 хв):
   - Автоматизація щотижневого звіту на реальних даних (Meta Ads + Google Ads)
   - Алерти по рекламним кабінетам (просадки балансу, критичні падіння CTR/CPA)
   - Google Merchant Center alerts (disapproved products → AI summary)
   - AI-візуал для креативного відділу (Nano Banana Pro, ChatGPT Images, Midjourney)
   - Огляд нових інструментів за місяць

## Артефакти учасника

- Один автономний сценарій тижневого звіту
- Налаштовані алерти (рекламний кабінет / Merchant Center)
- 1-2 згенеровані візуальні варіанти через AI
- Особистий 3-місячний роадмап
